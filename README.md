## 📊 Project Overview
This project applies **AI-driven predictive marketing** to The Coca-Cola Company. Using real financial data, we answer three core business questions:
1.  **Which ad channel gives the highest ROI?** (Linear Regression)
2.  **Can we forecast Q1 2026 revenue accurately?** (ARIMA Time Series)
3.  **Who are our customers?** (K-Means Segmentation)

## 🚀 Key Results
- **Growth:** Revenue increased +45.2% (2020-2025) coinciding with a shift to 70% Digital ad spend.
- **ROI Discovery:** Social Media generates **$4.10 per $1 spent** (2.7x higher than TV).
- **Forecast:** ARIMA predicts **$11.83B** for Q1 2026.
- **Segments:** Identified 3 distinct personas (Digital Natives, Value Seekers, Premium Champions).

## 🛠️ Models & Tools
- **Regression:** Statsmodels (OLS) - Achieved **R² = 0.896**
- **Forecasting:** ARIMA(2,1,1) - Using ACF/PACF plots
- **Clustering:** K-Means (Scikit-learn) - WCSS Inertia = 2300
- **Data Handling:** Pandas, NumPy, Matplotlib
